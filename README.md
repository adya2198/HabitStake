# HabitStake

## Project Overview
HabitStake is a productivity application designed to help users track and maintain their habits. It provides various features to enhance user engagement and motivation.

## Features
- **Habit Tracking**: Users can create, edit, and delete habits. The application tracks progress over time and allows users to set goals.
- **Reminders**: Users can set reminders for their habits to ensure they stay on track.
- **Statistics**: The application provides statistics and insights into user habits, helping them understand their patterns and improve productivity.

## Technical Specifications
- **Frontend**: Built using React.js for a dynamic and responsive user interface.
- **Backend**: Node.js and Express for the server, handling API requests and user authentication.
- **Database**: MongoDB for storing user data and habit information.
- **Deployment**: Deployed on Heroku for easy access and scaling.

## Usage Instructions
1. Clone the repository: `git clone https://github.com/adya2198/HabitStake`
2. Navigate to the project directory: `cd HabitStake`
3. Install dependencies: `npm install`
4. Start the application: `npm start`
5. Open your browser and go to `http://localhost:5173` to view the application.

For more detailed information, visit the project documentation or wiki.
